Welcome!

Mozart has challenged you to a piano duel!

Here is how to beat Mozart!

Mozart will start by playing some notes on his piano.

All you have to do is repeat what Mozart has played and you can beat him at his own game!

Each piano key matches a key on your computer keyboard.

  • C is set to the A Key
  • D is set to the S Key
  • E is set to the D Key
  • F is set to the F Key
  • G is set to the J key
  • A is set to the K Key
  • B is set to the L Key
